Graphing Function Transformations

1. Parent Function

The parent function is the simplest form of a function.

Example: f(x) = x²

2. Vertical Shift

Adding a number moves the graph up or down.

f(x) = x² + 2

👉 Moves the graph 2 units up

f(x) = x² - 3

👉 Moves the graph 3 units down

3. Horizontal Shift

Changing inside the function moves it left or right.

f(x) = (x - 2)²

👉 Moves the graph 2 units right

f(x) = (x + 1)²

👉 Moves the graph 1 unit left

4. Stretch and Compression

f(x) = 2x²

👉 Vertical stretch (narrower graph)

f(x) = 0.5x²

👉 Vertical compression (wider graph)

5. Reflection

f(x) = -x²

👉 Reflects across the x-axis

Summary

  • x² + c → up/down
  • (x - c)² → left/right
  • ax² → stretch/compress
  • -x² → reflection
